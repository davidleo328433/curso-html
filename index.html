tablas en html

en este apartado veremos como hacer tablas en las paginas web. principalmente se hace con la etiqueta table donde todo lo que este dentro seran los elementos de una tabla
mediante la etiqueta tr (table rows)indicaremos los renglones de la tabla, con las etiquedas de th(table head) podemos poner titulos a los renglones
con la etiqueda de td( table date) podremos agregar datos estandares dentro de nuestra tabla
al agregar otro tr estaras agregando otra fila con la cantidad de columnas que hayas asignado mediante th
dandole a la etiqueta table propiedad de style podras asignar espacio entre los datos
con ayuda del lenguaje css fuera del body se puede aplicar este estilo a cada elemento de la tabla aplicando una etiqueta de nombre style y mencionando que para la tabla, para th y para td se le asignaran esas caracteristicas
poniendo las propiedades dentro de llaves
dentro de la etiqueta table podemos utilizar una proiedad para hacer mas grande nuestra tabla mediante la propiedad style
usando thead y tbody podremos indicar cuales son las cabeceras y cual es el cuerpo de nuestra tabla.
